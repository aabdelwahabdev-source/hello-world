# WebCraft Project

Generated with WebCraft AI

## How to Run
This project includes an interactive 
simulation in the Console tab.

### Available Commands
- **`ls`**: List all generated files.
- **`cat [file]`**: View file content (e.g. `cat server.js`).
- **`docker-compose up`**: Simulate full-stack deployment (builds & runs containers).
- **`docker-compose down`**: Stop and remove containers.
- **`node server.js`**: Run the backend server only.

## Docker Setup
1. Define services using Docker blocks.
2. Drag a **Server** block for backend logic.
3. Switch to **Console** tab and type `docker-compose up`.
